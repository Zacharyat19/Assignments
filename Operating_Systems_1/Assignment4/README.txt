How to run:
    The program is compiled with (gcc --std=gnu99 -pthread -o line_processor main.c)
    This will create an executable that can run as (./line_processor)
    Will run with valgrind as (valgrind ./line_processor)
    Will run with ./line_processor < input.txt or ./line_processor < input.txt > output.txt
